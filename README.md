## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/your-username/FocusPulse.git

# Navigate to project
cd FocusPulse

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
